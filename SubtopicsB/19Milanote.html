Milanote

     Milanote is a visual collaboration and organization tool developed by Ollie Campbell and the Milanote team, officially launched in 2017. Designed primarily for creatives such as designers, writers, marketers, and teams working on visual projects, Milanote combines the structure of a productivity app with the flexibility of a digital whiteboard. It was developed to give users a space to collect ideas, plan projects, and see their creative process laid out visually in one place.

     Milanote’s purpose is to help users organize thoughts, inspiration, and project content in a way that mirrors how creative minds work—visually and spatially. It allows users to gather notes, images, links, files, and to-do lists into a flexible drag-and-drop interface, creating moodboards, storyboards, and project briefs with ease. Features include visual boards, checklists, rich text editing, web clippings, real-time collaboration, and private or shared workspaces. Milanote supports team collaboration through comments, shared boards, and real-time syncing. Its interface encourages nonlinear thinking and is particularly suited for brainstorming, creative planning, and ideation. With integrations for images, links, and file uploads, Milanote offers a central place to plan and visualize creative work from concept to completion. It’s ideal for individuals or teams looking to manage projects in a flexible, visually engaging environment.

User Interface

Milanote Home

     The Milanote Home UI is a visual workspace designed for organizing ideas. At the top is the header with navigation, settings, and sharing options. The left sidebar features tools like Note, Link, To-do, Line, Board, Column, Comment, Table, Image, Upload, and Draw. The center area is a canvas where users can add and arrange content freely. In the image, a board named “Jerome” is centered, representing a container for cards or notes. The UI promotes drag-and-drop functionality for intuitive organization. On the top-right, there's a counter for unsorted content. The overall design emphasizes simplicity, flexibility, and visual clarity for brainstorming, planning, and collaboration.

Milanote Board View

          Board View in Milanote provides a flexible, visual workspace for organizing ideas, tasks, and content. It features an infinite canvas where users can freely drag and drop notes, images, links, to-do lists, and more. Items can be grouped using columns or sub-boards, and connected with lines to show relationships. The left toolbar offers tools to add content, while the top bar includes options to share, export, or change views. A template panel helps users quickly start with pre-made layouts like moodboards or project plans. Board View is ideal for creative workflows, project planning, brainstorming, and collaboration. Its freeform structure supports both high-level strategy and detailed organization, giving users full control over how their information is displayed and linked.
