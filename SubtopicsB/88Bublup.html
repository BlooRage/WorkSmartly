Bublup

      Bublup is an all-in-one tool that stores any type of documents, notes, photos, and even videos. The tool allowed us to share them between teams privately. The tool also makes it easy to create private spaces for a wide range of uses. The tool was founded in 2014 in Arlington, Virginia, creating Bublup to address the challenges of digital clutter and disorganized information storage. The tool's name is a play on the word bubble up, meaning to reflect it's mission to surface important content in an organized manner. Bublup's user friendly interface allows user to create visual folders where they can store various types of content, including documents, photos, videos, and links. These folders can be customized with images and descriptions, making it easier to identify and access stored items.

      One of Bublup's standout features is it's ability to create "rolls", which are visually appealing web pages generated from folder content. This feature is particularly useful for sharing portfolios, project summaries, or curated collections with others. Bublup also supports collaboration by allowing users to invite others to their folders, facilitating teamwork and information sharing. with its combination of storage, organization, and sharing capabilities, Bublup serves as a versatile tool for both personal and professional use. By choosing Bublup, users can streamline their digital lives, making it easier to store, organize, and share information in a secure and visually engaging environment.

User Interface

Bublup Dashboard

      After logging in, users are greeted by the My Stuff tab. At the top, you’ll find intuitive navigation buttons for accessing your items, tasks, messages, and profile.Just below are filter options like Recent, Favorites, Shared with Me, and Emails to help refine visible content quickly. The main area displays all folders and items, and clicking on a folder allows users to dive deeper into its contents with a smooth, visual experience. The floating blue plus (+) icon opens the “Create New” menu, where users can easily add folders, save web links, jot down notes, create task lists, start Google Docs or forums, and upload files, images, or entire folders from their own computer. Users can also drag and drop content for quick organization. Each item or folder can be customized with images, titles, and descriptions, making the dashboard both functional and visually engaging.

Bublup Tasks Space

      The Bublup Tasks screen is the productivity hub where users can organize, track, and manage their to-do lists efficiently. The left panel offers smart task views such as Today, Upcoming, Overdue, Search, and Assigned to Me. The central area displays the selected task list, where users can easily check off completed items or click to expand them for more details. The blue “Add Tasks” button (plus icon) allows users to quickly add new tasks. Below each task, users can engage with icons to comment, like, or attach images, making collaboration and personalization seamless. The circular refresh icon in the top - right corner ensures that task updates are always synced in real time. Users can also drag and reorder tasks for prioritization, set reminders or due dates, and group tasks into folders for better organization.This layout makes it simple to streamline your workflow and daily responsibilities.
