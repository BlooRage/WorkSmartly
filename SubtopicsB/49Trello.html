Trello

User Interface

Trello Dashboard

     The Trello UI design highlights uniformity and usability in the dark human centered manner. The left sidebar allows fast access to boards, templates, and workspace settings. Popular templates such as Basic Board or Kanban Template, are positioned centrally to help users start quickly. Incorporations with JIRA are advertised with the visual cards located just below the templates. The Recently viewed section provides quick access to the last opened board. The bottom panel gives users access to all workspaces and the possibility to create a new board. Icons are clearly legible, minimalistic and intuitive Facilitating access for easy navigation. The interface above facilitates a clean workflow through strong visual grouping thereby allowing even a newbie or seasoned user to navigate easily and efficiently. Transitions between menus are smooth and responsive, reducing friction in user interactions.
